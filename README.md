# Expense Tracker (OOP)

A simple command-line Expense Tracker built with Python using Object-Oriented Programming (OOP).

This project was created to practice Python fundamentals, object-oriented design, input validation, and clean code organization.

## Features

- Add a new expense
- Show all expenses
- Search expenses by title
- Calculate total expenses
- Calculate average expense
- Delete an expense with confirmation
- Input validation for numbers and dates

## Project Structure

### Expense

Represents a single expense with:

- Title
- Amount
- Date

### ExpenseTracker

Manages a collection of expenses and provides operations such as:

- Add
- Show
- Search
- Calculate total
- Calculate average
- Delete

## Technologies

## How to Run

Clone the repository:

```bash
git clone https://github.com/AidaHaghdadi/expense-tracker-OOP.git
```

Go to the project directory:

```bash
cd expense-tracker-OOP
```

Run the program:

```bash
python main.py
```
